Configuration Highlights

Here are some of the key components of my dotfiles setup. Each configuration is designed to work together to create a cohesive development environment.

Terminal Setup

Alacritty terminal with Catppuccin Mocha theme and CaskaydiaCove Nerd Font

~/.config/alacritty/alacritty.yml

Shell Configuration

Zsh with Powerlevel10k prompt, custom aliases, and environment variables

~/.zshrc

Neovim Setup

Fully configured Neovim with LSP, treesitter, and Catppuccin colorscheme

~/.config/nvim/init.lua

Tmux Configuration

Terminal multiplexer with status bar customization and keybindings

~/.tmux.conf

Git Configuration

Custom git aliases, difftool/mergetool setup, and global ignores

~/.gitconfig

Installation

To use these dotfiles, clone the repository and run the installation script:

git clone https://github.com/yourusername/dotfiles.git ~/.dotfiles
                        cd ~/.dotfiles
                        ./install.sh

Pro Tip

Before running the install script, back up your existing dotfiles. The script will prompt you before overwriting any existing configurations.
